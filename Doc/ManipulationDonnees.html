Guide : Manipulation des données : datasets et transformations.

Dans cette librairie, les données sont transmises au réseau neuronal grâce à une structure de données spéciale : le Tensor. Il s’agit d’un tableau multidimensionnel : par exemple, un vecteur est un Tensor de rang 1 et une matrice un Tensor de rang 2. Cependant, afin de ne pas multiplier le nombre de paramètres à passer à chaque fonction de la bibliothèque, il existe une structure faite pour être passée à la fonction d’apprentissage des réseaux neuronaux : le Dataset.

Utilisation de la structure Dataset

Cette structure contient principalement quatre objets de type Tensor, de la manière suivante :

  • training_labels : Etiquettes pour les données d’apprentissage
  • training_data : Données d’apprentissage
  • test_labels : Etiquettes pour les données de test
  • test_data : Données de test.
Ainsi, il suffit de passer cette structure en paramètre à la fonction train d’un réseau neuronal pour l’entraîner, et toutes les données d’entraînement sont regroupées en un objet.

Les transformations

Lorsqu’on récolte des données que l’on va ensuite utiliser avec des algorithmes de deep learning, il peut être intéressant d’appliquer certaines modifications aux données avant leur utilisation. On parle, en anglais, de pre-processing. Par exemple, si on travaille avec des images, il est parfois avantageux de ramener la valeur de chaque composante de chaque pixel à l’intervalle [0, 1].

C’est pourquoi la librairie implémente les transformations. Il s’agit d’une fonction appliquée à chaque Tensor du Dataset. Par exemple, si nous souhaitons transformer chaque donnée de training_data et de test_data de l’intervalle [0;255] à l’intervalle [0; 1], nous pouvons utiliser le code suivant :

my_dataset.add_data_transforms(Divide(255.0)); // Ajoute la transformation souhaitée
my_dataset.apply_data_transforms(); // Applique les transformations qui ont été ajoutées

Pour plus d’informations sur les transformations existantes et sur la procédure à suivre pour créer vos propres transformations : page sur les transformations.
